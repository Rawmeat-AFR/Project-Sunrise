# 一.**概要**

本章は、日来世界の言語における音韻・拍構造の確定事項を記述する。

---
# 二.**母音体系**

母音は五種とする。長短対立を採用し、長音はマクロン付き母音で表記する。

|母音|長音表記|
|---|---|
|a|ā|
|i|ī|
|u|ū|
|e|ē|
|o|ō|

---
# 三.**子音体系**

子音は十三種とする。/p/ は採用しない。/r/ は日本語的弾き音（ɾ）として扱う。

|種別|子音|
|---|---|
|鼻音|m / n|
|破裂音（無声）|t / k|
|破裂音（有声）|b / d / g|
|摩擦音|s / h / z|
|半母音|w / y|
|流音|r（ɾ）|

有声破裂音（b / d / g）は語頭制約により語頭に現れない。

---
# 四.**拍単位**

基本拍単位は (C)V（子音＋母音、または母音単独）とする。以下の**特殊モーラ**を採用する。

|種別|内容|表記例|
|---|---|---|
|長音|母音を一拍延ばす|ka → kā|
|撥音|音節末の鼻音（独立一拍）|kan.na|
|促音|子音の重複（一拍の間）|lt表記（例：altka）|

---
# 五.**音韻制約**

|制約|内容|
|---|---|
|語頭制約|有声破裂音（b / d / g）の語頭出現を禁止する|
|語末制約|語末に置けるのは母音・撥音・長音のみ|
|子音連続禁止|子音の連続を禁止する。子音は必ず母音を挟む|

---
# 六.**表記規則**

## 六.一 撥音の環境依存分布

撥音は後続音の環境によって六種に変化する。

|撥音|出現環境|
|---|---|
|m|後続が b / m|
|n|後続が t / d / s / z / r / n（歯茎音）|
|ɲ|後続が tɕ / dʑ / ɲ / y / i|
|ŋ|後続が k / g / ŋ|
|ɴ|語末|
|Ṽ（鼻母音化）|後続が母音 / w / s / z / h|

## 六.二 その他の表記規則

|種別|表記|備考|
|---|---|---|
|長音|ā / ī / ū / ē / ō|マクロン付き母音|
|促音|lt|固定表記|
|有声破裂音|b / d / g|語中・語末のみ出現|
|半母音|w / y||
|弾き音|r|ɾの転写|
