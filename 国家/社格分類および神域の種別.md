# 一.**概要**

本資料は、日来世界における神社の社格分類および神域の種別・管理体制を記述する。 社格は祭祀規模と祭祀を怠った場合の影響範囲を基準として定められる。 神域は性質によって一種・二種に、大きさによって一類から三類に分類される。

---

# 二.**社格分類**

## 二.一 分類体系

社格は通常六段階で構成される。神宮および指定神社は通常の段階とは独立した特別枠として設けられる。
```
【特別枠】
  神宮　　　　大社の上位。有力信仰または多数の分社を持つ社に指定
  指定神社　　影響範囲が大きいが祭祀規模が伴わない社に指定

【通常社格】（上位→下位）
  1. 大社
  2. 域社
  3. 郷社
  4. 里社
  5. 祠
```

|社格|祭祀規模|祭祀を怠った場合の影響範囲|
|---|---|---|
|神宮|最大規模|国家・広域|
|大社|最大規模|国家・広域|
|域社|大規模|複数領域|
|郷社|中規模|一領域|
|里社|小規模|一集落・一区域|
|祠|最小規模|局所的|

## 二.二 神宮

神宮は大社の上位に位置する特別な社格である。 以下のいずれかの条件を満たす社が神務院の審査を経て指定される。

- 特定地方において有力な信仰を有する
- 多数の分社を持つ

神宮の指定は神務院による審査を経て行われ、条件を失った場合は指定が取り消されることがある。 鴉峨神宮はその代表的な事例であり、八嶋諸邦において最も古い神宮指定を受けた社とされる。

## 二.三 指定神社

指定神社は、祭祀規模と影響範囲が釣り合わない社に対して設けられる特別枠である。 祭祀規模は小さくとも、祭祀を怠った場合の影響範囲が大社・域社に匹敵する場合、神務院の判断によって指定される。

指定を受けた社には以下の措置が講じられる。

- 神務院による優先的な支援
- 祭祀体制の整備に向けた人員・資材の配置
- 定期的な神務院の監査

指定神社は祭祀体制が整い次第、通常の社格審査を受けることが求められる。 審査の結果、相応の社格が認定された場合、指定は解除される。

## 二.四 分社

分社とは、本社の影響範囲に祭祀が行き届かない場合に設置される社である。 設置の判断は神務院が行い、神託とは無関係に制度的必要性から設けられる。

分社の社格は本社とは独立して審査されるが、本社の社格を上回ることはない。

```
本社（例：域社）
  ├─ 分社A（郷社以下）
  ├─ 分社B（郷社以下）
  └─ 分社C（郷社以下）
```

分社は本社の祭祀体制を補完する役割を担うが、独立した祭祀主体として機能する。 本社が廃社となった場合、分社の存続については神務院が個別に判断する。

# 三.**神域分類**

## 三.一 概要

神域とは、神社およびその周辺において特別な管理が施される区域である。 神域は性質によって一種・二種に分類され、大きさによって一類から三類に細分される。 神域への無断侵入は法的罰則の対象となる。

```
神域の分類軸

  性質
  ├─ 一種神域　祭祀保護を目的とする区域
  └─ 二種神域　御神体そのものと一致する区域

  大きさ
  ├─ 一類　小規模
  ├─ 二類　中規模
  └─ 三類　大規模
```

## 三.二 一種神域

祭祀保護を目的として設けられる神域である。 人が立ち入ることで祭祀が妨害されるおそれのある区域に指定される。

主な指定対象は以下の通りである。

- 神社の本殿・拝殿周辺の核心区域
- 祭祀に用いる水源・火源が存在する区域
- 祭祀の執行上、外部の干渉を排除する必要がある区域

標準的な管理体制のもとに置かれ、神社の祠官が管理責任を負う。 区域の境界は注連縄・結界石・標柱などによって明示される。

## 三.三 二種神域

御神体そのものが神域と一致する区域である。 山全体・森全体・湖全体など、御神体の規模が大きく神域と御神体が不可分である場合に指定される。

立ち入りは御神体への直接干渉にあたるため、一種神域より厳格な管理体制が敷かれる。 二種神域の管理は神務院が直接関与し、祠官のみの管理に委ねられない。

主な指定対象の例は以下の通りである。

- 御神体として指定された山岳の全域
- 御神体として指定された森林・原野の全域
- 御神体として指定された河川・湖沼の水域

## 三.四 大きさによる細分類

一種・二種いずれの神域も、大きさによって以下の三類に細分される。

|類別|規模の目安|管理体制|
|---|---|---|
|一類|小規模（神社境内程度）|祠官による自主管理|
|二類|中規模（集落周辺に及ぶ程度）|祠官および地域の領・区による共同管理|
|三類|大規模（山岳・広域に及ぶ程度）|神務院の直接関与のもとでの管理|

## 三.五 侵入に対する罰則

一種・二種を問わず、神域への無断侵入は法的罰則の対象となる。 二種神域への侵入は御神体への直接干渉にあたるため、一種神域への侵入より重い罰則が科される。

罰則の軽重は以下の要素を総合して判断される。

- 一種・二種の別
- 類別（一類〜三類）
- 侵入の意図（過失・故意の別）
- 侵入による祭祀への実害の有無

罰則の詳細については別途規定する。
