# 1.**概要**

本規定は、過疎化等により維持不能となった神格を安全に解体し、土地を自然へ還流させるための実務工程を定めたものである。本工程の核心は、依代（または実体の御神体）が形成する「自爆回避領域」を物理的な作業基地として活用し、再利用可能な人工物を救出した後、人為的に災害を誘発させて神性を消費し尽くすことにある。

---
# 2.**実務工程**

## 第1段階：住民の避難と選抜体制の構築

人的被害をゼロにし、かつ作業時間を確保するための段階である。

1. 終了告知と意識の管理（齋禮官）：
    地域住民へ神格の終了を告知する。この際、住民の神格に対する感情（未練や感謝）が神性を不規則に励起させないよう、「土地の公的整理」としての側面を強調し、意識の沈静化を図る。
    
2. 一般住民の退避：
    作業に関与しない全住民を、予測される災害射程圏外へ移動させる。
    
3. 選抜祭祀集団の残留：
    解体・搬出が終わるまで神格を「和御魂」に繋ぎ止めるため、最小限の齋禮官および住民作業員が、依代の加護が及ぶ安全圏（爆心地）に宿営する。
    

## 第2段階：依代加護下の資源救出（搬出工程）

依代周辺の「物理的に守られた空間」を作業拠点とし、土地から不純物（人工物）を回収する。

1. 人工物の計画的解体と搬出：
    住宅、公共施設、道路舗装、地下配管に至るまで、将来の復興を阻害する人工物をすべて解体する。解体された資材は、依代の加護が及ぶ範囲内に一旦集積し、順次、安全圏外の集積所へと輸送する。
    
2. 神性濃度の常時監視（封厄官）：
    搬出作業中、封厄官は依代周辺および搬出路の神性濃度を定時測定する。神格が「臨界点」に近づき、先行して微細な霧や異常現象が発生していないかを確認し、作業員の安全を物理・精神の両面から確保する。
    
3. 維持祭祀の継続（齋禮官）：
    齋禮官は維持祭祀を行い、神格の和御魂を保ち続ける。
    

## 第3段階：荒御魂への強制転換と災害の解放

全ての搬出が完了したことを確認し、安全かつ確実に神性を消費させる最終段階である。

1. 一斉退去：
    全ての資材と人員が、依代の加護（安全圏）を離れ、災害想定域の外へ完全に脱出する。依代（または実体の御神体）のみが更地に取り残された状態を作る。
    
2. 荒御魂化祭祀：
    人員の安全を確認後、齋禮官はこれまでの維持祭祀を一転させ、意図的に荒御魂を刺激する祭祀を行う。これにより、自然な崩壊を待たずに「災害の発生」を人為的に早め、制御下でエネルギーを放出させる。
    
3. 霧化顕現の処理（封厄官）：
    災害の解放と同時に、大量の霧化顕現が発生する。待機していた封厄官は、この霧が生存圏へ流出しないよう、火砲等を用いて物理的に霧を晴らし続ける。
    
4. 還土の確認：
    
    災害が収まり、霧が消散した後、神性が「潜在神性（LV-0）」まで低下したことを確認し、土地の自然還流（および無人地としての管理）を宣言する。

# 4. 齋禮官と封厄官の役割分担

|担当|主な実務内容|責任|
|---|---|---|
|齋禮官（主要）|維持祭祀による時間稼ぎ、災害発生の制御、住民の意識誘導。|神格が予定より早く暴走することのないよう、精神的な「蓋」を管理する責任。|
|封厄官（補佐）|依代周辺の神性濃度確認、作業中の安全確保、災害発生時の霧化顕現の強制的消散。|物理・精神汚染が作業員や生存圏に及ばないよう、実力で空間を清浄化する責任。|

# 5. 禁止事項

- 依代の早期移動・破壊： 加護（安全圏）を消失させ、解体作業員を死に追いやるため厳禁とする。

- 資材の現地放置：* 人工物が自然に混ざることは、土地の「死」を意味する。釘一本、凝土片一つ残さぬ回収が求められる。

- 無計画な解放： 封厄官の準備が整わない状態での荒御魂化は、広域な霧化顕現災害を招くため、齋禮官の独断での実行は禁ずる。
