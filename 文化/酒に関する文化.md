# 1.**概要**

日来世界における酒は、単なる嗜好品ではなく、神格に対する感謝と人の楽しみが不可分に結びついた農耕文化の一部である。

---
# 2.**民俗学的文化**

古来から続く文化要素を基盤とした作法が生活に浸透している。

## 2.1 指し水（さしみず）

* 定義：飲酒の終盤、最後の一杯に水を加える行為。  
* 意図：泥酔による理性の喪失（無礼）を避け、水という自然の形に自我を近づけることで、自律を祈る。

## 2.2 献杯と廻し盃

1. 献杯：まず、神格に酒を供えるための専用の盃で祠や分社の御神札に酒を捧げる。  
2. 廻し盃：次に、別の盃を参列者全員で順に回し飲む。  
* 機能：一つの盃を共有することで、場の「概念」と集団の意志を統一し、厄を分散・除去する連帯の儀式とする。

## 2.3 庭酒（にわさけ）：感謝の循環

* 作法：家庭の祠の前で、酒を少量を地面に撒く。  
* 機能：自然から借りたものを独占せず、人為を加えた酒を還流させることで、自然の恵みを循環させる。

## 2.4 ハレの日の「無礼講」

* 厳格な日常の規律から一時的に解放され、賑やかに酒を飲み、笑い合う。

# 3.**酒の分類定義**

## 3.1 原材料による大分類

* 純米系（じゅんまいけい）：米、米麹、水のみ。米本来の旨味やコクが強く、添加物を一切含まない。  
* 本醸造系（ほんじょうぞうけい）：米、米麹、水、および醸造アルコール。香りを引き出し、整える技術。

## 3.2 精米歩合による等級分類

米をどの程度まで磨き（削り）、雑味を取り除いたかによる分類。

|    精米歩合    | 米・米麹（純米系） | 米・米麹・醸造アルコール（本醸造系） |
| :--------: | :-------- | :----------------- |
|   50%以下    | 純米大吟醸酒    | 大吟醸酒               |
|   60%以下    | 純米吟醸酒     | 吟醸酒                |
| 60%以下/特別製法 | 特別純米酒     | 特別本醸造酒             |
|   70%以下    | （純米酒）     | 本醸造酒               |

## 3.3 製法・状態による副次分類

* 生酒：もろみを取っただけの酒。  
* 生貯蔵酒：生のまま貯蔵し、出荷時に一度だけ火入れしたもの。  
* 生詰酒：熟した酒を火入れせず瓶詰めし出荷したもの。  
* 生一本：自醸酒で純米酒のもの。  
* 原酒：加水調整をしていないアルコール度数が高い酒。  
* おり酒：下呑から抽出したおりが多く含まれるもの。  
* 長期貯蔵酒：数年以上貯蔵された古酒。  
* 樽酒：樽の木の香りを生かした酒。  
* にごり酒：醪を目の粗い布でこした白濁酒。  
* 発泡酒：炭酸ガスを含み、アルコール分は8度程のもの。

## 3.4 品質評価の要件

* 米麹割合：使用米の総重量に対し、米麹重量が15%以上であること。  
* 醸造アルコール量：白米重量の10%以下であること。  
* 等級審査：各地域の醸造組合または当局による品質検査を通過していること。

## 3.5 市場流通上の名称（普通酒）

基準に該当しないものは「普通酒」として日常的に広く消費される。

## 3.6祭祀的分類（御神酒）

* 御神酒（おみき）：祭祀において、依り代の前に捧げられる酒。捧げられた瞬間に「嗜好品」から「祭具」へと転換される。  
* 直会（なおらい）：祭祀の後に人間がこれを下ろし、いただく行為。
