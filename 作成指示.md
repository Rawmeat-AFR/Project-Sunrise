# 2.**書式**
## 2.1 見出し記述法

- 大見出し：# n.**大見出し**
- 中見出し：## n.n 中見出し
- 小見出し：### 小見出し
注1：中見出し以下は上位の見出し内に設けること
注2：nには見出し番号を入れること。ただし小見出しは不要。
注3：各大見出しの間には区切り線(---)を入れること。

## 2.2 改行

- 本文：改行は各見出しと区切り線から一行開けるようにせよ。
- 見出し：見出し同士は改行をしなくてよい。

## 2.3 太字

太字は本文中で重要箇所のみ使用せよ。
例外：大見出しのみ常に使用。
